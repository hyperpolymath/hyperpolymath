== Architectural Rigour: The 6SCM Layer

Every repository contains a machine-readable Scheme layer in `.machine_readable/6scm/` to ensure AI agents and human collaborators remain synchronised.

[horizontal]
`STATE.scm`:: Roadmap and blockers.
`META.scm`:: ADRs and architectural rationale.
`AGENTIC.scm`:: AI interaction patterns and safety boundaries.
`NEUROSYM.scm`:: Symbolic reasoning rules.
